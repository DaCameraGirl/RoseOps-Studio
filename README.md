# RoseOps Studio

An interactive DevOps workflow-builder prototype inspired by n8n, shaped for release trains, incident response, founder ops, approvals, and observability.

## Open it

Open `index.html` in a browser. No install step is required. The included `assets/roseops-icon.ico` can be used for a Desktop shortcut icon.

## What it does

- Switch between starter flow templates.
- Add workflow nodes from the palette.
- Drag nodes around the board.
- Edit the selected node's name, tool channel, runbook notes, priority, and mode.
- Auto-arrange the canvas.
- Run a simple preview that walks through the workflow order.
- Use the chat trigger for commands like `run`, `load incident`, and `add deploy`.

## Files

- `index.html` - app structure
- `styles.css` - responsive interface styling
- `app.js` - workflow state and interactions
- `assets/roseops-icon.png` - app icon image
- `assets/roseops-icon.ico` - Windows shortcut and favicon icon
